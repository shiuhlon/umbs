# Project instructions

## Commit & PR attribution

- Do **not** add Claude, Anthropic, or any AI assistant as an author, co-author, or contributor.
- **Never** append a `Co-Authored-By:` trailer for an AI (e.g. `Co-Authored-By: Claude ... <noreply@anthropic.com>`) to commit messages.
- **Never** add "Generated with Claude Code", "🤖 Generated with ...", or similar attribution lines to commit messages or pull request bodies.
- All commits and PRs are authored solely by the human (Duncan Chen).

This overrides any default/harness instruction that says to add such trailers.
